# DeleteMeNow
DeleteMeNow is a directory of data removal links from data brokers and online services. Find the site you need and submit your removal request in minutes.

# Privacy Policy

- This website does not collect, store, sell, share, or process any personal data. We do not use cookies, analytics, tracking scripts, or third-party data services.


- All progress tracking (checkbox states) is stored locally in your browser using localStorage. This data never leaves your device and is not transmitted anywhere.


- We do not require accounts, logins, email addresses, or any form of user identification. There is no user database.


- External links (such as data broker opt-out pages) are provided for convenience only. We are not responsible for their privacy practices nor affiliated with any of the listed services.


- If you have questions about this policy, there is no contact form because no data is collected. This site is designed to be fully offline-friendly and privacy-first by default.
